Adds an image_src meta tag to your header using post thumbnail - helps sites like FaceBook use a relevant image as thumbnail when sharing content.

== Description ==
This plugin helps ensure your content has a relevant thumbnail when shared on social sites. If your articles have no image associated with them when shared on social sites, or worse, if your RSS icon keeps showing up as the thumbnail this plugin will help you define the correct image to be used. 

Adds an image_src meta tag to your header using post thumbnnail. You must have post thumbnails / featured image enabled in your theme for this plugin to be effective. 

== Installation ==
1. Upload `add-image-src-meta-tag.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
