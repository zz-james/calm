This plugin simplifies embedding of Issuu publications in blog posts. Simply copy/paste the publication URL from your browser into your blog post.

== Description ==

This plugin is now deprecated. Please use the new official [Issuu embed](http://wordpress.org/plugins/issuu-embed) plugin instead.

== Installation ==

Here's how to install the plugin and how to make it work.

1. Download and activate the plugin through the Plugins menu in WordPress admin
2. Copy Issuu publication URL from your browser and just paste it into the body of your post. Example: http://issuu.com/iscience/docs/issue23
3. If you are the owner of the publication on Issuu, you can use the link with the URL parameter e=xxxx/yyyy, which you get when sharing from the My Publications section on your Issuu account , to make use of styling options.

Make sure that the URL is on a separate line in your blog post.

== Changelog ==

`Version 3.1`

Plugin deprecated. Please use the new official [Issuu embed](http://wordpress.org/plugins/issuu-embed) plugin instead.

`Version 3.0`

Complete plugin rewrite. This version basically adds Issuu to the internal whitelist of oEmbed providers to allow the Wordpress oEmbed support to use issuu.com

`Version 2.15`

Fixed a bug with force default width for all embeds. Added support for parameter `folderId`

`Version 2.10`

Added support for Enterprise documents. Added support for parameter `proSidebarEnabled`

`Version 2.01`

Added support for Issuu Reader 2.0

`Version 1.31`

Fixed a bug with *undefined offset* error. Added support for parameter `proshowmenu`, `proshowsidebar`

`Version 1.21`

Fixed a bug with the html hyperlinks. Added support for parameter `showflipbtn`, `folderid`

`Version 1.2`

Added three useful hyperlinks below the embed viewer, which are *Open publication*, *Free publishing*, and *More*.

`Version 1.1`

This version includes significant improvements, which enables the user to embed full size viewer. It is compatible with the WordPress.com embed code that Issuu provides for each publication.
